# Запуск тестов для API
1) Создать venv: python -m venv env;
2) Запустить env: перейти в папку с env → scripts → activate;
3) Обновить pip: pip install -U pip;
4) Установить библиотеки: pip install -r requirements.txt;
5) Запуск тестов pytest tests;
6) В папке a yandex можно проверить любой url, по умолчанию проверяется яндекс. Запуск для проверки нужного url: pytest test_yandex --url=нужный url
